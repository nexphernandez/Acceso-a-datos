# Starter — Servicios Spring sin REST (H2)

Este starter está **sin implementación** y **sin tests**. El objetivo es que completes:
- Las anotaciones JPA en `AlumnoEntity`.
- Los métodos del adaptador `AlumnoRepositoryJpaAdapter`.
- La lógica del servicio en `AlumnoService`.

## Requisitos
- Java 17+, Maven 3.8+

## Ejecutar (arranca, pero fallará donde haya TODOs)
```bash
mvn spring-boot:run -Dspring-boot.run.profiles=h2
```
